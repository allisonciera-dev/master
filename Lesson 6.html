Lesson 6: CSS Dimensions and Pseudo-Classes

CSS lets you control the size and visibility of HTML elements using properties like width, height, and visibility. These help you design layouts that look good and work well on different screens.

Dimension Properties

  • width: sets the width of an element (e.g., width: 300px)
  • height: sets the height (e.g., height: auto)
  • line-height: controls spacing between lines
  • max-height / min-height: limits the height range
  • max-width / min-width: limits the width range
  • visibility: controls whether an element is visible, hidden, or collapsed

Pseudo-Classes

Pseudo-classes are special CSS selectors that apply styles based on user interaction. They’re written with a colon (:) and used mostly with links.

  • :link – styles unvisited links
  • :visited – styles visited links
  • :hover – styles when mouse hovers over
  • :active – styles when link is clicked
  • :focus – styles when element is focused

Remember the order: LVHA – Link, Visited, Hover, Active.

What I learnt?

I learned that CSS can do more than just change colors—it can hide stuff, set sizes, and even make links change when you hover or click them. I didn’t know there was a special order for link styles, like LVHA. That’s kinda cool because it makes websites feel alive when you move your mouse around!
