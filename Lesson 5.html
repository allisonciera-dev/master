Lesson 5: CSS Positioning

1. Introduction to CSS Positioning

The position property in CSS determines how elements are placed on a webpage. It helps designers control the layout and create organized, responsive pages. There are five main types of positioning: static, relative, absolute, fixed, and sticky.

2. Types of CSS Positioning

  • Static – The default position. Elements are placed in the normal flow of the document.
  • Relative – Moves the element relative to its normal position using properties like top, bottom, left, and right.
  • Absolute – Positions the element relative to its nearest positioned ancestor. It is removed from the normal document flow.
  • Fixed – Stays in the same position even when scrolling the page, often used for navigation bars.
  • Sticky – Acts like relative positioning until the element reaches a specific point on the page, then becomes fixed.

3. Float and Clear Properties

The float property is used to move elements to the left or right side of a container. It is often used to place text beside images. The clear property prevents elements from floating next to others, forcing them to appear below floated elements.

img {
  float: left;
  margin-right: 10px;
}

p {
  clear: both;
}

4. Flexbox Layout

Flexbox is a modern layout system that makes it easier to align and distribute space among items inside a container. It helps create flexible and responsive designs.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Flexbox is especially useful for centering content and organizing elements in rows or columns.

5. Grid Layout

The CSS Grid layout allows designers to create two-dimensional layouts. It uses rows and columns to arrange items neatly.

.container {
  display: grid;
  grid-template-columns: auto auto auto;
  gap: 10px;
}

Grid layouts are powerful for building structured webpages like dashboards or galleries.

6. Summary

CSS positioning and layout properties help control how elements appear on the screen. Using position, float, flexbox, and grid gives web designers flexibility and precision when organizing content.

7. What I Learned
