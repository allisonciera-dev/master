Lesson 3: CSS Basics, Display, Span, and Div Tags

1. Types of CSS

There are three main types of Cascading Style Sheets (CSS):

  • Inline CSS – Applied directly within an HTML tag using the style attribute.
  • Internal or Embedded CSS – Written inside the <style> tag within the <head> section of the HTML file.
  • External CSS – Written in a separate .css file and linked using the <link> tag.

2. The Display Property

The display property controls how an element is shown on a webpage. The two main display types are:

  • Block-level elements – Occupy the full width available and start on a new line (e.g., <p>, <div>, <section>).
  • Inline elements – Display within a line of text without breaking it (e.g., <span>, <a>, <em>).

Common values of the display property include: display: block;, display: inline;, display: list-item;, display: none;, and display: inherit;.

3. The Div Tag

The <div> tag divides a webpage into sections. It is a block-level element often used to group related content or define layout areas like headers and sidebars.

<div class="header">Welcome!</div>

4. The Span Tag

The <span> tag is an inline element used to style or highlight specific text within a line. It does not have default formatting, so CSS must be used to style it.

<p>This is a <span style="color:red;">highlighted</span> word.</p>

5. Difference Between Div and Span

  • Div – Used for large sections or layout blocks.
  • Span – Used for small portions of text or inline styling.

6. Summary

CSS helps define the design and layout of a webpage. Understanding the display property, as well as the div and span tags, allows better control of webpage structure and presentation.

7. What I Learned

In this lesson, I learned how CSS helps make a webpage look more organized and creative. I now understand how to use the div tag to divide sections and the span tag to style small parts of text. I also found out how the display property affects how things appear on the page. It’s really cool to see how a few lines of code can change the whole look of a website.
