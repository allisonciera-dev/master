Lesson 4: CSS Rules and Classes

1. CSS Rules

A CSS rule defines how HTML elements should look or behave. Each rule has two main parts: a selector and a declaration block.

  • Selector – Targets the HTML element you want to style (e.g., p, div, h1).
  • Declaration Block – Contains one or more declarations separated by semicolons. Each declaration has a property and a value (e.g., color: blue;).

Example of a CSS rule:

p {
  color: blue;
  font-size: 16px;
}

2. CSS Selectors

Selectors are used to choose the specific HTML element(s) you want to style. The most common types are:

  • Type Selector – Styles all elements of a specific tag (e.g., p).
  • Class Selector – Styles elements with a specific class name, written with a dot (e.g., .highlight).
  • ID Selector – Styles a single element with a unique ID, written with a hash symbol (e.g., #main).
  • Universal Selector – Styles all elements on the page (e.g., * {margin:0;}).

3. CSS Classes

Classes are used to style multiple elements with the same appearance. To define a class, use a period (.) followed by the class name in your CSS. Then apply it to any HTML element using the class attribute.

.highlight {
  color: red;
  font-weight: bold;
}

<p class="highlight">This is a highlighted paragraph.</p>

4. CSS IDs

IDs are similar to classes but are meant to be unique — only used once per page. To define an ID, use a hash (#) before its name.

#header {
  background-color: lightblue;
  padding: 10px;
}

<div id="header">Welcome!</div>

5. Comments in CSS

Comments are notes in your code that don’t affect how it runs. They are written inside /* ... */.

/* This is a comment */

6. Summary

CSS rules and classes are the foundation of styling in web design. Selectors help target specific elements, while classes and IDs make it easier to apply consistent styles throughout your page.

7. What I Learned

I understand how selectors can target certain parts of a webpage and how I can use classes to style many things at once. I also learned the difference between a class and an ID, which I think is really useful for web design. This lesson helped me realize how CSS makes everything look professional and neat.
